# 一、为什么要学习变换：

> 变换是计算机图形学的基石，旋转、缩放、平移、投影等等操作，甚至从3D到2D的成像，本质上都是在进行变换。

# 二、二维空间的变换（线性变换）：
## 1. 线性变换的数学概念：
- **变换(transformations)** $T:V \rightarrow W$**，也叫函数(functions)或映射(mapping)**，规定了从**定义域**(domain) $V$ 到**对应域(codomain)** $W$ 的对应关系，对于 $V$ 中的每个元素 $x$， $W$ 中都有唯一元素 $T(x)$ 和它对应，$W$ 中那些被对应到的元素组成的子集叫做**值域**(range)，值域可以是整个 $W$。$T(x)$ 叫做 $x$ 的像(image)， $x$ 叫做 $T(x)$ 的原像。
- 在标准的 $n$ 维欧几里得空间（如 2D 平面或 3D 空间）中，一个变换 $T$ 要被称为**线性变换**，必须同时满足两个条件（也是其的定义）：
	1. **加法性**：$T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})$
	2. **齐次性**：$T(c\mathbf{u}) = cT(\mathbf{u})$
- 由此可以推导出一个必然结论：**线性变换必须保持原点不变**，即 $T(\mathbf{0}) = \mathbf{0}$。
	- 证明：$T(\mathbf{0}) = T(0 \cdot \mathbf{u}) = 0 \cdot T(\mathbf{u}) = \mathbf{0}$。
-  $T(x)=Ax+b$ 当 $b \neq 0$ 时，不满足$T(\mathbf{0}) = \mathbf{0}$，可知其不是线性变换。这个线性+平移的变换我们称作**仿射变换(affine transformations)**，在下面会详细讲解。

# 三：齐次坐标
## 1. 什么是齐次坐标：
![[Pasted image 20260314145726.png]]
- 让我们观察一下“**平移**”变换：你会发现，当你把方程组写出来，想将其转换为矩阵的时候，其无法使用上面已有的线性变换矩阵方程进行描述，必须使用向量加法表示：$y=x+b$ 
- 如果你想对一个物体先旋转、再平移、最后缩放，你需要混合使用乘法和加法：$y=S(R(x))+ty=S(R(x))+t$
	- 这种混合运算在计算机硬件（特别是GPU）上效率较低，且难以将多个变换合并为一个简单的操作。
- 那么，有没有一个统一的方式对所有变换进行描述呢？
- 这就引出了“**齐次坐标**”的概念：
![[Pasted image 20260314145811.png]]
- 发现没：当我们将二维向量添加一个维度，拓展到三维时，平移操作就可以写成一个矩阵乘以一个向量的形式了，现在，所有几何变换都可以统一写作$Y = \text{M}X$的形式了！
![[Pasted image 20260314145859.png]]
- **为什么齐次坐标要有点和向量的区别**：
	- 通过最后一个分量 $w$，齐次坐标可以明确区分”点“（有位置）和”向量“（只有方向）
	- 当 $w = 1$ 时，乘以平移矩阵时 $1\cdot t_x$ 生效，点的位置被改变
	- 而当 $w = 0$ 时，乘以平移矩阵时 $0\cdot t_x = 0$ ，平移量被抵消，向量本身保持不变
	- 通过 $w$ 分量的区别，齐次坐标简便地实现了向量不变性

- **点 + 点的解释**：两个点相加，$w = 2$，结合下面的解释，将 $w$ 化为1，则该坐标就是**这两个点连线的中点**的齐次坐标
- **最后一点的解释**：若$w \neq 0$，我们可以将分量同除 $w$，也就是说，$(x,y,w)$表示$(x/w, y/w, 1)$ 这个二维平面上的点

# 3. 复合变换：
![[Pasted image 20260314151029.png]]
- 复合变换需要注意顺序：矩阵乘法是不满足交换律的！
![[Pasted image 20260314151103.png]]
- **理解**：
	- 因为我们默认使用列向量，设这个向量为 $v$，我们要先做变换 $A$，得到中间结果 $v'=A(v)$，再对中间结果 $v'$ 做变换 $B$，得到最终结果$v''=B(v')$
	- 将其连起来写就是函数的**复合**：$v''=B(A(v))$
	- 如果将变换函数写作矩阵乘法形式（如$v'=Av$），则可转换为 $v''=B(Av)$
	- 由于矩阵乘法的结合律，可以转换为如下形式：$v''=(BA)v$
	- **这里举个栗子：** 设矩阵为 $4\times4$ 大小，变换次数为 $K$ 次，整个图形有 $N$ 个点需要变换：
		- **一步一步计算矩阵$\times$向量**，每个点的每个变换（即 $v'=A(v)$）需要16次乘法和12次加法，整个计算的总代价 $\text{Total} = N \times K \times 28$ ，复杂度为 $O(N \times K)$
		- 但如果**先计算总变换矩阵，再乘以向量**，每次矩阵相乘只需要64次乘法和48次加法，最后再计算1次矩阵$\times$向量，整个计算的总代价  $\text{Total} = (K-1) \times 28 + (N\times28)$，复杂度为 $O(K + N)$
		- 很明显，复杂度从乘法降到了加法，大幅下降 $\uparrow$
	- ![[Pasted image 20260314152519.png]]
	- 其实就是这个意思 $\uparrow$，也就是说，无论有多少层嵌套变换，只要它们都是线性的（仿射），我们就可以通过矩阵乘法结合律，把所有变换‘折叠’成一个单一的总矩阵，从而让每个顶点只需经历一次矩阵-向量乘法。

## 补充：
- ![[Pasted image 20260315154503.png]]
- 绕原点旋转 $-\theta$ 的变换矩阵是绕原点旋转 $\theta$ 的变换矩阵的**转置矩阵**，同时也是其的**逆矩阵**
- 换到线性代数的概念可知，绕原点旋转 $-\theta$ 的变换矩阵是**正交矩阵**，因为有$R_{-\theta}=R_\theta^T = R_\theta^{-1}$ ，且$R_\theta^TR_\theta^{-1}=R_\theta^{-1}R_\theta^T=I$, $I$ 为单位矩阵。
	- 在数学上如果**一个矩阵的逆等于它的转置**，那我们管这个矩阵叫做**正交矩阵**。
