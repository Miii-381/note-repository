> **反走样**（Anti-aliasing），也称**抗锯齿**，是一种**减少或消除“锯齿”（jaggies）现象的技术**。锯齿通常出现在图形边缘，尤其是在斜线或多边形边界处，由于像素是离散的方形单元，无法完美表示连续曲线或斜线，从而造成视觉上的阶梯状瑕疵。反走样通过在边缘像素上进行颜色混合（例如根据覆盖比例混合前景与背景色），使边缘看起来更平滑。

> **深度缓冲**（Z-buffer）是一种用于解决“**可见性问题**”的技术，即确定哪些物体或表面在场景中是可见的，哪些被其他物体遮挡.
> 每个像素除了颜色信息外，还保存一个“**深度值”**（Z 值），表示该像素距离摄像机的远近。在渲染过程中，新绘制的像素会与其对应的深度缓冲值比较：如果新像素更近，则更新颜色和深度值；否则丢弃该像素。这种方法确保了即使绘制顺序混乱，也能正确显示前后关系，避免“穿透”或“重叠错误”。

- 由于像素的物理性质和图形采样的局限性，会导致最终显示到屏幕上的图像出现“锯齿状”瑕疵（**走样**，Aliasing）。

- 采样的局限性是广泛存在的，这里用一个词表示 —— Artifacts：![[Pasted image 20260402150726.png]]
- 采样的局限性体现在很多地方，包括**图形的锯齿、摩尔纹、车轮效应、果冻效应**等等...![[Pasted image 20260402150907.png]]
- 为了解决走样，我们会采用**反走样**技术，对图像进行进一步处理。
- 反走样的操作无外乎两步——**先模糊，再采样**。其**核心**在于在采样前对原始图像做**模糊（预过滤/滤波）**![[Pasted image 20260402151151.png]]
![[Pasted image 20260402151250.png]]
- **简单来说就是对清晰的信号进行发散（滤波\模糊），使得原本锐利的边缘呈现出平滑过渡的效果，对模糊后的信号进行采样的时候，就避免了采样二值化导致的锯齿效应**
- 可以看看下面的图，非常直观地展示了反走样对锯齿的优化效果：![[Pasted image 20260402151945.png]]
- 那么，我们能不能先采样，再对采样后的图像进行过滤呢？答案是不行的！
- 对于先采样后模糊，在采样时，图像**原始的高频信息**（精细的格子纹理）已经永久丢失，不可避免地**劣化**出了锯齿，图像的根本结构已经发生了改变；
- 后期处理算法通常是在屏幕上找“颜色反差大的边缘”，然后把这些边缘涂模糊，根据这种算法处理后，只会得到**模糊的锯齿**——锯齿的形状还在，只是边缘被晕开了，整体画面看起来会很脏、很糊。![[Pasted image 20260402152311.png]]
- 但 yysy ，这种操作还是存在的，比如 FXAA 和 TAA ，虽然效果差了点，但性能较 MSAA 和 SSAA 这种多重采样有极大提升，算是一种为了性能而妥协画面的折中操作。

### 🔍 用傅里叶变换解释采样、走样和反走样

它正是那把能让我们从数学层面“看见”图像频率，并精确理解为什么“先模糊再采样”才是正解的钥匙。你可以把它看作是连接图像空间与频率空间的桥梁。

简单来说，傅里叶变换的核心思想是：**任何复杂的波形（或信号），都可以拆解成无数个不同频率、不同振幅的正弦波的叠加。**

在图形学中，这就好比把一张复杂的图片，拆解成了“**低频部分**”（平滑渐变、大色块）和“**高频部分**”（锐利边缘、噪点、纹理细节）。

如果把图像看作是二维信号，傅里叶变换就是把图像从**空间域**转换到**频率域**。

- **空间域**：我们平时看到的图像。横轴是 x 坐标，纵轴是 y 坐标，颜色代表亮度。
- **频率域**：经过傅里叶变换后的图像。横轴和纵轴代表**频率**（u, v），亮度代表该频率成分的**强度**（振幅）。
#### 频率代表什么？

- **低频**：代表图像中变化缓慢的部分。比如一片蓝天、人物的皮肤、远处的山脉。
- **高频**：代表图像中变化剧烈的部分。比如物体的锐利边缘、栅栏的纹理、噪点、**锯齿**。

**举个通俗的例子：**  
如果你听一首交响乐（空间域信号），你的耳朵听到的是所有乐器混合在一起的声音。  
傅里叶变换就像是一个超级精准的乐谱分析员，他能告诉你：这首曲子里，有多少分贝的小提琴（高频），多少分贝的大提琴（中频），多少分贝的贝斯（低频）。

### 💻 傅里叶变换在图形学中的核心应用

回到我们刚才讨论的“反走样”，傅里叶变换是理解这一切的数学工具。

#### 解释“为什么会有锯齿”（采样定理）

这是傅里叶变换在图形学中最经典的应用。

- **原理**：在频率域中，**采样**会导致频谱的**周期性复制**。
- **走样**：如果原图包含太多的“高频信息”（比如密集的栅栏），而我们的采样率（屏幕分辨率）不够高，这些复制出来的频谱就会**互相重叠**。这种重叠在空间域的表现，就是**锯齿**和**摩尔纹**。
- **结论**：**奈奎斯特采样定理**告诉我们，采样频率必须至少是信号最高频率的 2 倍，才不会发生混叠。

#### 解释“模糊就是低通滤波”

之前提到的“预模糊”，在傅里叶变换的视角下变得非常直观：

- **卷积定理**：空间域的**卷积**（模糊操作），等于频率域的**乘积**。
- **低通滤波**：当我们对图像进行高斯模糊时，实际上是在频率域里，把代表细节和边缘的**高频部分**直接乘以 0（或者衰减掉），只保留**低频部分**。
- **反走样流程**：
    1. 原图（含高频） -> **傅里叶变换** -> 频谱（含高频峰）。
    2. **乘以低通滤波器**（切掉高频） -> 剩下的只有低频。
    3. **逆傅里叶变换** -> 变回空间域，此时图像变模糊了（预模糊）。
    4. **采样** -> 因为高频没了，频谱不会重叠，所以没有锯齿。

#### 图像处理与纹理压缩

- **JPEG 压缩**：JPEG 的核心就是离散余弦变换（DCT，傅里叶变换的亲戚）。它把图片转成频率域，然后发现人眼对高频不敏感，于是把高频数据狠狠压缩甚至丢弃，只保留低频，文件就变小了。
- **纹理各向异性过滤**：这也是为了解决频率不匹配的问题。

#### 程序化生成与噪声

- 很多自然纹理（云、火、大理石）是通过**柏林噪声**或**单纯形噪声**生成的，这些噪声函数的设计大量借鉴了频率合成的思想（把不同频率的噪声叠加起来，这叫分形布朗运动）。

-------
### 📌 总结

**傅里叶变换是图形学的“X光机”。**

- 它让我们看到：**锯齿 = 高频信号过剩**。
- 它告诉我们：**反走样 = 低通滤波（切除高频）**。
- 它也证明了：**先采样后模糊是没用的**，因为一旦频谱重叠（走样发生），高频和低频混在一起，就再也分不开了，怎么滤波都去不掉那个“脏”味。

> 在计算机图形学中，**滤波（filtering）** 是一种用于处理图像或信号的技术，而 **卷积（convolution）** 是实现滤波的核心数学操作之一。

### 什么是卷积？

卷积是一种数学运算，用于将一个函数（通常称为“核”或“滤波器”）应用于另一个函数（如图像），以产生一个经过修改的新函数。在离散图像处理中，图像被视为二维像素数组，卷积操作通过滑动一个小的矩阵（即卷积核）遍历整个图像，并在每个位置计算加权和。

### 卷积操作的基本步骤：

1. **定义卷积核（Kernel）**：一个小的二维矩阵，例如 3×3 或 5×5，其中每个元素代表一个权重。
2. **滑动核**：将核中心对准图像中的每一个像素。
3. **逐元素相乘并求和**：将核与图像对应区域的像素值逐元素相乘，然后将结果相加，得到输出图像中该位置的新像素值。
4. **边界处理**：对于靠近图像边缘的像素，可能需要采用填充（padding）策略（如零填充、镜像等）来处理缺失的邻域像素。

### 常见应用

- **模糊（Blurring）**：使用均值滤波器或高斯滤波器平滑图像。
- **锐化（Sharpening）**：增强图像边缘和细节。
- **边缘检测（Edge Detection）**：如 Sobel、Prewitt 或 Laplacian 算子。
- **抗锯齿（Anti-aliasing）**：在渲染过程中减少图像中的锯齿现象。
- **纹理合成与图像重建**：在高级图形应用中用于生成或修复图像内容。

### 数学表达（离散二维卷径）

对于图像 $I(x, y)$ 和卷积核 $K(u, v)$，卷积结果 $(I * K)(x, y)$ 定义为：

$$ (I * K)(x, y) = \sum_{u=-a}^{a} \sum_{v=-b}^{b} I(x+u, y+v) \cdot K(u, v) $$

其中 $(2a+1) \times (2b+1)$ 是核的尺寸。

> 总结来说，**卷积**是图形学中实现各种视觉效果和图像处理任务的**基础工具**，通过设计不同的**卷积核**，可以灵活地控制图像的局部特征变化。

		- 设有一张大小为 $N\times N$ 的图像，用 $K\times K$ 大小的卷积核进行卷积；
		- 直接在空域上进行卷积的计算量很大，每个像素需要 $K^2$ 次乘加操作，总操作数约为：$O(N^2K^2)$，当 $K$ 增大时，计算量会急剧上升
		- 而如果使用卷积定理进行计算，则复杂度能被压缩到 $O\left( N^2 \log N \right)$（这里没补零），在卷积核大的情况下非常划算。
		- 但注意，FFT的常数开销还是挺大的，在卷积核较小（$K$ 较小，比如$3\times3$、$5\times 5$）时其实不太划算，只适合卷积核较大的情况。
	- 注：GAMES101中演示的的都是**循环卷积**，也就是说，在做卷积时，当卷积核滑动到右边缘，**超出的部分不是补零或镜像，而是从另一侧“绕回来”**——就像在一个无限循环的环面上操作。实际应用中，我们通常通过**补零**将其转化为**线性卷积**

# 4. 采样（Sampling）
![[Pasted image 20260408165042.png]]![[Pasted image 20260408170040.png]]
- 解释一下：
	- 左边很简单，是函数值随时间变换的图像；
	- 右边是频谱图，横轴是频率 f，纵轴是**频谱幅度（或频谱密度）**，曲线下的面积代表该频段的分量强度；
		- **f 很大**：代表信号变化非常快，一会儿上一会儿下（比如锯齿波、高频噪声）。
		- **f = 0** ：代表信号**完全不变化**，是一个恒定的值。这在物理上被称为“直流分量”**。
	- 图像中出现的箭头其实是特殊的函数，叫做**冲激函数**，
		- **时域理解：** 想象它是一次**极其短暂且猛烈**的敲击。它在时间 $t=0$ 的瞬间值为无穷大，其余时间为 0。虽然它持续时间无穷短，但它包含的“总能量”（面积）是 1。
		- **频域理解：** 因为这次敲击太“猛烈”且“突然”，它包含了**所有频率**的成分，而且每个频率的强度都是一样的。所以，单个冲激的频谱是一条**平坦的直线**（也就是包含从 f=0 到无穷大的所有频率）。
	- 图 a 是原始函数，频谱图是图 b；可以看到，原始函数的频谱是一个小三角；
	- 图 c 是冲激函数构成的**冲激串**，其频谱图是图 d。
		- 图 d 中的每一个箭头，代表在这个特定的频率点（$0,f_s,2f_s...$ ）上有能量，而两个箭头之间没有任何能量。
		- 而因为冲激串是一个周期函数，根据**傅里叶级数**理论，它可以分解为无数个正弦波的叠加，公式如下：$脉冲串 = A_0 + A_1\sin(2\pi f_st) + A_2\sin(2\pi \cdot 2f_st) + A_3\sin(2\pi \cdot 3f_st) + ...$（冲激函数的展开系数 $A_n = 1/T$，$T$ 为冲激串的采样周期 ）
		- 关于高度 $1/T$：单看一个冲激脉冲，其每隔时间间隔 $T$ 出现一次，能量为1（这是积分定义），而由于平铺图的能量强度是平均能量，所以该函数在 $T$ 上的平均能量强度（直流分量、傅里叶系数）就为 $总强度/周期 = 1/T$ ；
		- 所以冲激串的频谱图是间隔为采样频率 $f_s = \frac{1}{T}$ 、能量强度为 $1/T$ 的箭头。
	- 将 a 与 c 相乘，也就是用冲激串去“筛选”原始连续函数，得到的是图 e。此时，图 e 不再是连续的波形，而是一串**强度被调制的冲激函数**。虽然它在时间上仍然是连续的（只是在非采样点为 0 ），但其**冲激的强度**（面积）恰好等于原始函数在采样点的数值。原始函数的曲线就像是这些冲激的**包络线**。而体现在频谱图上，就是图 f ，根据**时域相乘等于频域卷积**的性质，原本的小三角形频谱被“复制”了无数份，并以采样频率 $f_s$​ 为间隔，均匀地排列在横轴上。

- 这张图解释了数字信号处理的根本原理，同时也揭示了图形学中的**走样**（Aliasing）是如何发生的：

1. **频谱搬移**：只要你把模拟信号变成数字信号（采样），频谱就会在频率轴上不断重复。
2. **奈奎斯特采样定理**：注意看图 (f)，为了让原始信号不失真地恢复，这些复制的三角形（频谱）**不能重叠**。
    - 如果采样频率 $f_s​$ 足够高（大于原始信号最高频率 $f_0$​ 的 2 倍），频谱之间就有空隙，我们可以完美还原信号。
    - **如果采样太慢（ $f_s​$​ 太小）**：这些三角形就会挤在一起并发生**重叠**。一旦发生重叠，原本的信息就被破坏了，这就叫**频域混叠**，反映在图像上就是锯齿，反映在视频里就是车轮倒转。（**见下图**）

> 所以，上图就是在告诉你：**采样不仅仅是记录数值，它在本质上是在频域里复制并搬移信号的频率成分。**

- 在实际操作中，有一种非常使用的滤波器： ![[Pasted image 20260408180320.png]]
![[Pasted image 20260408180337.png]]
![[Pasted image 20260408180348.png]]
- 简单来说就是以图像在该像素内的占地面积作为整个像素的颜色权重，没有为0，占满为1。
- 这种通过**精确计算像素内几何图形的覆盖面积**进行反走样（解析反走样、area sampling）的方法，需要计算图像的积分得到图像面积，图像一旦变复杂了就极其难以实现，通常只作为其他反走样方法的理论支撑。
- 计算机中一般使用基于**离散点**的**多重采样抗锯齿**（MSAA，将像素拆分为多个子采样点进行加权平均）或**超采样抗锯齿**（SSAA，将整个画面以更高分辨率进行渲染，然后下采样到目标分辨率）等方法进行反走样。

- 当然，世界上没有免费的午餐，抗锯齿带来了平滑的画面，但代价是大幅提高了计算量...
- 为了优化计算量，在计算时一般会用到边缘检测、采样点复用等操作来减少需要计算的像素。
- 还有一些抗锯齿，是先采样后滤波的，也就是对图像进行后期处理，比如FXAA和TAA（当然，性能强了，图像质量就没那么好了）。
	- **FXAA（Fast Approximate Anti-Aliasing，快速近似抗锯齿）**：先采样生成图像，再通过一些图像匹配算法找到边缘，然后把这些边缘模糊/混合，从而实现锯齿的平滑。
	- **TAA（Temporal Anti-Aliasing，时间性抗锯齿）**：利用多帧历史信息，静态图像可以小幅移动采样点或摄像机，动态图像直接使用。将当前帧与之前几帧的对应像素进行混合（加权平均），得到新像素
- 还有个神奇的方法：**DLSS（Deep Learning Super Sampling，深度学习超级采样）**，它是先渲染一个小画面，再利用显卡上的深度神经网络模型，根据低分辨率图像进行分析，“脑补”出高分辨率图像。
